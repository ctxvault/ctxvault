# CtxVault v0.4.1 Release Notes

Status: experimental, non-normative design preview for the Projection
Governance Kernel.

## Positioning

v0.4.1 packages the Projection Governance Kernel as a docs/schema/fixture/test
design preview. It explains the object model behind reviewed context handoff:

```text
source evidence -> candidate context -> review decision -> projection -> receipt
```

The design preview makes the projection control point explicit. Raw evidence is
not durable truth, candidate context is not automatic permission, and receipts
record selected, omitted, blocked, written, and not-done states.

## Public Boundary

This is a design preview. It is not a stable external API and not runtime
behavior. It is not:

- shipped v0.4.0 behavior
- a stable external API
- runtime behavior
- runtime gate wiring
- Workbench review UX
- provider or model execution
- live package, registry, API, or command checking
- external memory adapters
- durable memory consolidation or automatic memory promotion
- hallucination prevention
- memory control
- benchmark superiority
- public beta readiness
- Memory OS or unified agent memory
